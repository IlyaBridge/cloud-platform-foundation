# Kubernetes configuration

Данный репозиторий содержит декларативные конфигурационные файлы для настройки Kubernetes-кластера.

## Состав репозитория

### ingress/
Манифесты для установки и настройки ingress-nginx контроллера, обеспечивающего HTTP-доступ к сервисам кластера.

### application/
Манифесты для деплоя тестового приложения (nginx), включая:
- Deployment
- Service
- Ingress

### monitoring/
Конфигурация системы мониторинга на базе kube-prometheus-stack:
- Prometheus
- Grafana
- Alertmanager
- node_exporter

Используется Helm chart kube-prometheus-stack с вынесенной конфигурацией в values.yaml.

## Назначение

Все компоненты Kubernetes-кластера могут быть развернуты автоматически путем применения данных конфигурационных файлов без необходимости ручной настройки.


## P.S.
Для настройки Kubernetes был подготовлен отдельный git-репозиторий с декларативными конфигурационными файлами. Репозиторий включает manifests для ingress-контроллера, тестового приложения и системы мониторинга на базе kube-prometheus-stack.
